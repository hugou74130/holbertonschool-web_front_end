# Image Assets (Placeholders)

This folder is intended to contain the high-resolution images used by the CSS Advanced project (Responsive Design practice).

## What you should do

1. **Download the images from Unsplash** (https://unsplash.com) using the search keywords listed below.
2. Save the images with the exact filenames listed in this folder.
3. Replace the placeholder files in this folder with the downloaded images.

> ⚠️ This repository does not include the actual high-resolution images due to licensing and download constraints.

## Required images (filenames)

### Content images (10)
- `pic-about-01.jpg` — a strong hero/about image (e.g., people collaborating, office space)
- `pic-work-01.jpg`, `pic-work-02.jpg`, `pic-work-03.jpg` — work-related scenes (desks, laptops, meetings, coding)
- `pic-article-01.jpg`, `pic-article-02.jpg`, `pic-article-03.jpg` — article/reading/writing related
- `pic-person-01.jpg`, `pic-person-02.jpg`, `pic-person-03.jpg` — portraits or people headshots

### Branding images (logos + favicon)
- `logo-1.png` (or `logo-01.png`)
- `logo-2.png` (or `logo-02.png`)
- `favicon.ico`

> If the project description provides specific logo/favicon links, download those and save with the above filenames.

## Suggested Unsplash search terms

- **Work**: "workspace", "office desk", "remote work", "team meeting"
- **About**: "team collaboration", "creative team", "startup office"
- **Article**: "writing", "blogging", "notebook", "reading articles"
- **Person**: "portrait", "headshot", "professional person"

## Notes

- Use high-resolution images (at least 1920px wide) for responsive design practice.
- Make sure the image file names match exactly as listed so HTML/CSS references work without modification.
