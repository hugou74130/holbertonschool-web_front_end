# 🎨 CSS Style Files

This directory contains individual CSS files for the CSS Advanced module exercises.

## Structure

Each numbered file (`XX-style.css`) corresponds to a specific exercise in the Holberton School CSS Advanced curriculum, covering topics such as:

- Custom properties (CSS variables)
- Flexbox and Grid layouts
- Transitions and animations
- Responsive design patterns
- Advanced selectors

## Usage

These stylesheets are linked from the HTML files in the parent directory. Each exercise builds upon the previous one.
