## 🧐 About <a name = "about"></a>

This repository contains the web front-end specialization projects from Holberton School France. It covers semantic HTML5, advanced CSS techniques, accessibility best practices, and responsive web design principles.

## 🏁 Getting Started <a name = "getting_started"></a>

### Prerequisites

- A modern web browser
- A text editor (VS Code recommended)

### Installing

Clone the repository:

```bash
git clone https://github.com/hugou74130/holbertonschool-web_front_end.git
cd holbertonschool-web_front_end
```

## 📦 Project Modules <a name = "modules"></a>

| Module | Description | Technologies |
|--------|-------------|--------------|
| `html_advanced/` | Semantic HTML5 structure, accessibility, SEO | HTML5 |
| `CSS_advanced/` | Advanced CSS styling, animations, responsive design | CSS3 |

## 🎈 Usage <a name = "usage"></a>

Open any `.html` file in your browser:

```bash
# Example: open the advanced homepage
open html_advanced/index.html
```
