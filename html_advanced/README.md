# 🌐 HTML Advanced

Advanced HTML5 exercises for semantic markup, accessibility, and SEO.

## Files

This directory contains progressive HTML exercises building a complete website:

| File | Description |
|------|-------------|
| `index.html` | Main homepage (final version) |
| `about.html` | About page |
| `contact.html` | Contact page |
| `latest_news.html` | Latest news page |
| `article.html` | Article page |
| `styleguide.html` | Style guide / component documentation |
| `*-index.html` | Progressive homepage iterations (0 through 36) |
| `*-styleguide.html` | Progressive styleguide iterations |
| `favicon.ico` / `favicon.png` | Site favicons |

## Key Concepts

- Semantic HTML5 elements (`header`, `nav`, `main`, `article`, `section`, `aside`, `footer`)
- Accessibility best practices (ARIA labels, alt text)
- SEO optimization (meta tags, structured data)
- Form elements and validation
- Responsive image handling

## Usage

Open any `.html` file in a browser:

```bash
open index.html
```
